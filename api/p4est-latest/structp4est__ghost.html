◆ ghosts

sc_array_t p4est_ghost::ghosts

An array of p4est_quadrant_t quadrants which make up the ghost layer around p4est.

Their piggy3 (cf. p4est_quadrant::p4est_quadrant_data) data member is filled with their owner's tree and local number (cumulative over trees). Quadrants are ordered in p4est_quadrant_compare_piggy order. These are quadrants inside the neighboring tree, i.e., p4est_quadrant_is_inside is true for the quadrant and the neighboring tree.

sc_array_t p4est_ghost::mirrors
